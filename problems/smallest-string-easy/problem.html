Smallest String in Lexicographic Order (easy)

  • 実行時間制限: 1 sec
  • メモリ制限: 512 MiB

問題文

$N$ 個の文字列が与えられます。これらの文字列を辞書順に並び変えたとき、先頭の文字列を出力してください。

なお、辞書順とは「単語が辞書に載っている順番」です。厳密には、文字列 $S, T$ の大小は以下のアルゴリズムで判定されます。

  1. $S, T$ のうち長さが短い文字列の長さを $L$ とする。$i = 1, 2, \dots, L$ について $S_i$ と $T_i$ が一致するか調べる。
  2. $S_i \ne T_i$ である $i$ が存在する場合、そのような $i$ のうち最小のものを $j$ とする。 $S_j, T_j$ を比較し、 $S_j$ がアルファベット順で $T_j$ より小さい場合は $S < T$、大きい場合は $S > T$ とする。
  3. $S_i \ne T_i$ である $i$ が存在しない場合、$S, T$ の長さを比較して、 $S$ が $T$ より短い場合は $S < T$、長い場合 $S > T$ とする。

制約

  • $1 \le N \le 1000$
  • 各文字列の長さは 100 を超えない
  • 同じ文字列が含まれることはない
  • 文字列にはアルファベット小文字のみが含まれる

入力

入力は以下の形式で標準入力から与えられる。

$N$
$s_1$
$s_2$
$\vdots$
$s_N$

出力

辞書順に並び替えたとき、先頭の文字列を出力せよ。

入出力例

例1

入力

3
abc
bca
cab

出力

abc

End of Problem
